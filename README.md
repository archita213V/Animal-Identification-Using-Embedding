# Animal Identification using Computer Vision

## 📌 Overview

This project identifies animals from images using Computer Vision and Machine Learning techniques.

## 🚀 Features

- Animal detection from images
- Image segmentation
- Feature extraction
- Similarity search using FAISS
- Fast and accurate predictions

## 🛠️ Technologies Used

- Python
- OpenCV
- NumPy
- PyTorch
- SAM (Segment Anything Model)
- FAISS
- Jupyter Notebook

## 📂 Workflow

1. Input Image
2. Image Segmentation using SAM
3. Feature Extraction
4. Similarity Search using FAISS
5. Animal Prediction
